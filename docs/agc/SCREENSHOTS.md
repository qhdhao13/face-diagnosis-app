# AGC 应用介绍截图尺寸说明

HarmonyOS AGC 对截图尺寸要求较严，**必须与控制台所选设备类型、方向一致**，且同一组 3 张尺寸必须完全相同。

## 手机（竖屏）

| 项目 | 值 |
|------|-----|
| 目录 | `docs/agc/screenshots/` |
| 尺寸 | **450 × 800** 像素 |
| 数量 | 3 张（推荐 01、04、05） |
| 格式 | PNG，单张 ≤ 2MB |

## PC/2in1（横屏）

| 项目 | 值 |
|------|-----|
| 目录 | `docs/agc/screenshots-pc/` |
| 尺寸 | **1280 × 720** 像素（16:9） |
| 数量 | 3 张（01、02、03） |
| 格式 | PNG，单张 ≤ 2MB |

若仍提示尺寸不符，在 AGC 上传页查看该设备类型旁显示的**推荐分辨率**，把报错原文发开发者调整。

## 重新生成

```bash
python3 scripts/generate-agc-screenshots.py
python3 scripts/generate-agc-screenshots-pc.py
```

## 上传注意

1. **手机截图**上传到「手机」应用介绍截图（竖向）
2. **PC 截图**上传到「PC/2in1」应用介绍截图（横向）
3. 不要混用竖屏图到 PC 栏位，或横屏图到手机栏位
4. 三张必须同尺寸、同方向
